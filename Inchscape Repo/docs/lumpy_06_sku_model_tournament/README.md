# Lumpy 06 SKU Model Tournament

This folder is generated by `notebooks/lumpy_06_sku_model_tournament.ipynb`.

The notebook evaluates the full-history lumpy SKU population only, using an 18-month test window, a 3-month operational lag, and SKU-level average 3-month rolling WMAPE.

Main files:

- `lumpy_06_full_history_sku_model_scores.csv`: one row per SKU per model with SKU-level rolling WMAPE, horizon WMAPE, bias, positive-month counts, and totals.
- `lumpy_06_full_history_recommended_model_per_sku.csv`: one row per SKU with the best model selected for that SKU.
- `lumpy_06_full_history_model_win_summary.csv`: counts of how often each model wins.
- `lumpy_06_full_history_lowest_50_wmape_skus.csv`: the 50 easiest SKUs by selected rolling WMAPE.
- `lumpy_06_full_history_highest_50_wmape_skus.csv`: the 50 hardest SKUs by selected rolling WMAPE.
- `lumpy_06_full_history_recommendation_quality_summary.csv`: SKU counts by recommendation quality band.

Large raw monthly backtest rows are written to `results/lumpy_06_sku_model_tournament/`, which should stay out of Git.
