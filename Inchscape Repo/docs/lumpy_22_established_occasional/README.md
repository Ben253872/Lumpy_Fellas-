# Established Occasional A/B/C Tournament

Notebook 22 applies the official 18-month horizon, 3-month information gap, and six 3-month block metric to all 116 established occasional SKUs.

Models may differ by A/B/C, but every champion is selected on the same historical six-block validation horizon and locked before the final period is scored.

Main outputs:

- `lumpy_22_locked_strategies.json`
- `lumpy_22_official_summary.csv`
- `lumpy_22_individual_sku_results.csv`
- `lumpy_22_validation_single_candidates.csv`
- `lumpy_22_diagnostic_oracle_ceiling.csv`
- `lumpy_22_actual_vs_forecast.png`
