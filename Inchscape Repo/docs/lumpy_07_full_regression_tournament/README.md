# Lumpy 07 Full Regression Tournament

This folder is generated by `notebooks/lumpy_07_full_regression_tournament.ipynb`.

The notebook evaluates the full-history lumpy SKU population using an 18-month test window, a 3-month operational lag, and SKU-level average 3-month rolling WMAPE. It reframes the problem as direct and hurdle regression with SKU fixed-effect features.

Main files:

- `lumpy_07_full_history_regression_sku_model_scores.csv`: one row per SKU per regression model with rolling WMAPE, horizon WMAPE, bias, positive-month counts, feature count, and fit status.
- `lumpy_07_full_history_regression_recommended_model_per_sku.csv`: one row per SKU with the best regression model selected for that SKU.
- `lumpy_07_full_history_regression_model_win_summary.csv`: counts of how often each regression model wins.
- `lumpy_07_full_history_regression_quality_summary.csv`: SKU counts by recommendation quality band.
- `lumpy_07_full_history_regression_selected_fold_summary.csv`: monthly total fold summary for the selected regression models.
- `lumpy_07_vs_06_sku_recommendation_comparison.csv`: SKU-level comparison versus the previous `06` recommendation file when available.
- `lumpy_07_vs_06_summary.csv`: compact summary of whether regression improved versus `06`.

Large raw forecast files and checkpoints are written to `results/lumpy_07_full_regression_tournament/`, which should stay out of Git.
