# Lumpy 36 - New-SKU Level Calibration And High-Volume Routing

Notebook 36 retains Notebook 35's 75% TSB plus 25% subfamily analogue blend and tests historically selected global scales and recent-demand routing for the 39 new SKUs.

## Result

Historical validation selected a conservative 1.05 global scale rather than a
high-volume routing rule. On the official 39 new SKUs, below-70 coverage improved
from 13 to 16. Below-50 remained 2 and below-100 remained 31. Median WMAPE
improved from 74.84% to 74.19%, portfolio WMAPE from 79.93% to 79.62%, and bias
from -49.75% to -47.23%.

Notebook 36 is promoted as the new-SKU champion. The remaining portfolio bias is
concentrated in high-volume misses, but stronger historical scaling or routing
did not win selection. Further new-SKU tuning should wait for richer launch,
supersession, fitment, stock, or order-pipeline signals.
