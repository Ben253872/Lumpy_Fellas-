# Lumpy 08 Champion Model Tuning

Generated by `notebooks/lumpy_08_champion_model_tuning.ipynb`.

This notebook builds a final SKU-level champion table by combining the `06` intermittent-demand tournament, clean `07` regression results, and inner-validation tuned capped regression results.

Main files:

- `lumpy_08_existing_06_07_clean_champion_per_sku.csv`: fast champion using existing `06` plus clean `07`.
- `lumpy_08_tuning_validation_scores.csv`: inner validation scores showing which parameter candidates were selected per outer fold.
- `lumpy_08_tuned_capped_regression_sku_model_scores.csv`: one row per SKU per tuned capped regression model.
- `lumpy_08_tuned_capped_regression_recommended_model_per_sku.csv`: best tuned capped regression model per SKU.
- `lumpy_08_final_champion_candidate_scores.csv`: all candidate rows used for final model choice.
- `lumpy_08_final_champion_per_sku.csv`: final best available model per SKU.
- `lumpy_08_final_champion_summary.csv`: model/source win summary.
- `lumpy_08_final_champion_segment_summary.csv`: performance by predictability segment.
- `lumpy_08_final_champion_vs_06_summary.csv`: compact comparison back to `06`.

Large raw tuned forecasts and checkpoints are written to `results/lumpy_08_champion_model_tuning/`, which should stay out of Git.
