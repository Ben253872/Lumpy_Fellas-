# Lumpy 09 Croston XGBoost Hybrid

Generated by `notebooks/lumpy_09_croston_xgboost_hybrid.ipynb`.

This notebook tests the hybrid intermittent-demand approach: Croston/SBA/TSB projections become features, a classifier estimates demand occurrence, a regressor estimates positive demand size, and a tuned probability gate determines whether demand is expected.

Main files:

- `lumpy_09_hybrid_threshold_validation_scores.csv`: inner validation threshold scores by fold and model.
- `lumpy_09_hybrid_sku_model_scores.csv`: one row per SKU per hybrid model.
- `lumpy_09_hybrid_recommended_model_per_sku.csv`: best hybrid model per SKU.
- `lumpy_09_hybrid_coverage_audit.csv`: coverage check before hybrid models are allowed into the champion table.
- `lumpy_09_final_candidate_scores.csv`: fair `08` champion plus eligible `09` hybrid candidates.
- `lumpy_09_final_champion_per_sku.csv`: final best model per SKU after comparing fair `08` vs `09`.
- `lumpy_09_final_champion_summary.csv`: final source/model win summary.
- `lumpy_09_final_champion_vs_08_summary.csv`: compact comparison back to fair `08`.

Large raw hybrid forecasts and checkpoints are written to `results/lumpy_09_croston_xgboost_hybrid/`, which should stay out of Git.
